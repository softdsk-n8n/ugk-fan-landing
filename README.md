# UGK — Underground Kingz fan archive

Неофициальный фан-лендинг про UGK (Pimp C + Bun B, Port Arthur, TX) на русском.
Построен на **Astro + Tailwind v3 + vanilla JS** в brutalist-эстетике южного хип-хопа.

> ⚠ **Unofficial fan page.** All names, images, and music are property of UGK / Rap-A-Lot / Jive / respective rightsholders. Made with respect for the legacy.

## Запуск

```bash
npm install
npm run dev          # → http://localhost:4321
```

## Production build

```bash
npm run build        # → ./dist/
npm run preview
```
