# HANDOFF — UGK fan landing

## Статус

- ✅ Full-page brutalist fan landing (9 секций)
- ✅ Responsive (mobile + tablet + desktop)
- ✅ A11y baseline: alt, focus, reduced-motion
- ✅ SEO: meta, OpenGraph, Schema.org MusicGroup
- ⚠ 11 placeholders для изображений

## Быстрый запуск

```bash
npm install
npm run dev
```

## Deploy

### Vercel
```bash
npx vercel --prod
```

### Netlify
```bash
npm run build
npx netlify deploy --dir=dist --prod
```
